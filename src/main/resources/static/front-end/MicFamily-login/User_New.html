  • Welcome to  麦家-MicFamily

    系统介绍

    麦家-MicFamily是一个为量贩式KTV开发的运营管理系统，能够满足量贩式KTV除“歌曲点播”外的所有管理需求。MicFamily分为管理员端和用户端。管理员端拥有“KTV员工管理、KTV会员管理、包间管理、订单管理、商品管理、公告管理”6个功能模块。用户端拥有“订单查询、包间预订、结算、超市”4个功能模块”。在MicFamily的加持下，我们相信量贩式KTV能够实现管理的“零差错”，盈利的“高增长”，用户的“很满意”。

    技术介绍

    1. 前端：排版框架Bootstrap v5.1.0开发\ACE、字体与图标使用elementUI或font-awesome v4.7.0、jquery v3.5.1、VUE脚本、
    2. 后端：SprintBoot v2.7.1
    3. 数据库：mysql8.0.26
    4. 开发工具：后端与构建IDEA、前端界面HBuilderX、前端界面DreamWeaver、数据库Navicat 15.0.26
    5. 项目构建与版本管理：apache-maven-3.6.3
    6. 开发语言：java1.8

    功能介绍

    麦家MicFamily在管理员端实现了“员工管理、会员管理、包间管理、订单管理、商品管理、公告管理”6个模块的增删改查，支持用户的注册、登录、退出、上传头像操作；支持公告的下载操作；在主页左下角实时显示近期发生的待处理事务消息。在用户端实现了“订单查询、包间预订、在线结算、超市选购”4个功能，支持用户的登录、注册和退出登录；按包间类型查找包间，按包间号查询订单，按包间号在线下单。 项目使用云数据库，可在腾讯云图上反映KTV的实时运营情况。
